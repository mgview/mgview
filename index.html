MGView

A common problem with rigid-body simulations is that the output is a stream of numbers; interpreting the results through 2-D plots can be quite cumbersome. Using the power of HTML5 and WebGL, we can visualize the motion of 3D dynamic systems in the browser.
Due to non-uniform feature implementation by browser vendors, currently only Google Chrome is fully supported.
